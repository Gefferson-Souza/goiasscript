## 🚀 Como Usar

### Pré-requisitos
```bash
# Se ainda não tem o GoiásScript instalado globalmente:
npm install -g goiasscript
```

### Executar o projeto
```bash
goiasscript bota_pra_moer src/main.gs
```

### Construir (traduzir) o projeto  
```bash
goiasscript traduz src/main.gs
```

### Verificar tipos
```bash
goiasscript vê_se_tá_certo src/main.gs
```

### Debug
```bash
gs-fuça fuça src/main.gs
```

### REPL (Roda de Prosa)
```bash
goias
```

### Executar testes
```bash
goiasscript bota_pra_moer tests/test.gs
```

## 📚 Comandos GoiásScript

- `goiasscript traduz arquivo.gs` - Traduz GoiásScript para JavaScript
- `goiasscript bota_pra_moer arquivo.gs` - Executa arquivo GoiásScript
- `goiasscript vê_se_tá_certo arquivo.gs` - Verifica tipos
- `gs-fuça fuça arquivo.gs` - Debug interativo
- `goias` - REPL (Roda de Prosa)
- `gs-balaio mostra_os_balaio` - Gerenciar dependências

## 🔧 Desenvolvimento

1. **Criar nova funcionalidade**: Edite `src/main.gs`
2. **Testar**: Execute `npm run dev`
3. **Verificar**: Execute `npm run check` antes de commitar
4. **Debug**: Use `npm run debug` se algo não estiver funcionando
